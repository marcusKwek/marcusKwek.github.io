Arm Mounted Mouse

Project Dates: 12/2024 - 03/2025

Status: Completed

Tools involved: 3D Printing, C++, Python, Arduino, Onshape


Features:

● Laptop uses camera to track the position of the device with computer vision, and then refines the position using bounding boxes to allow for smooth, controlled movement of the cursor

● Stepper Motors move the jaws of the gauntlet to clamp around the user's arm. An ultrasonic sensor is used to detect the presence of the user and closes to the appropriate tightness

● An IMU tracks rotational motion to control features like clicking and scrolling

● A Bluetooth module is used to communicate between the laptop and device

● All parts of the device's frame were modelled in Onshape and 3D printed from PLA
